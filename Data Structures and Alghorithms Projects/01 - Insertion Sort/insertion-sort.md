# Insertion Sort Proje

``` 
[22,27,16,2,18,6] -> Insertion Sort
```

## Insertion Sort
---
Yukarı verilen dizinin sort türüne göre aşamalarını yazınız:
```
[22,27,16,2,18,6]
[2,27,16,22,18,6]
[2,6,16,22,18,27]
[2,6,16,18,22,27]
```

## Big-O Notation
---
Big-O gösterimini yazınız.
O(n^2)

## Time Complexity
---

Ortada olduğu için average case kapsamına girecektir.
